# Privacy Policy – S7 Image & Data Grabber

S7 Image & Data Grabber does not collect, store, transmit, or share personal data by default.

The application operates locally on the user’s device and processes only data and files explicitly provided by the user. Any images, files, or data handled by the application remain under the user’s control.

The application does not:
- Collect personal information
- Track user behavior
- Transmit data to third-party servers
- Use analytics or advertising services

If optional online features are enabled in future versions, this policy will be updated accordingly.
